PC Setup Instructions

install git
	https://git-scm.com/download/win
	download, install... (ask for instructor help with install options)

install python
	https://www.python.org/downloads/release/python-3109/
	download, install... (ask for instructor help with install options)

open command prompt
	press: win+r
	type: cmd
	press: enter

create github directory
	execute command: mkdir github

traverse to github directory
	execute command: cd github

clone GCTA repository
	execute command: git clone https://github.com/GCTA-Python/GCTA-Python-Intro.git

install python virtual enviornment
	execute command: python -m venv venv
	...if that does not work, try:
		python3 -m venv venv
		py -m venv venv

activate virtual environment
	execute command: ./venv/Scripts/activate

install jupyter notebook
	execute command: pip install notebook

launch jupyter notebook
	execute command: jupyter notebook
